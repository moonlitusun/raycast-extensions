# Antigravity Search Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] - {PR_MERGE_DATE}

### Added

- Initial release of Antigravity Search extension
- Search and open projects in Antigravity IDE
- Configurable projects directory
- Adjustable search depth (1-5 levels)
- Custom Antigravity command path support
- Automatic filtering of common build folders (node_modules, dist, etc.)
- Multiple actions: Open with Antigravity, Show in Finder, Copy Path
- Display last modified time for projects
